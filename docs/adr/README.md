# Architecture Decision Records (ADR)

ADR ghi lại các quyết định kiến trúc quan trọng: ngữ cảnh, lựa chọn và hậu quả.

**Format mỗi ADR** (khuyến nghị):

- **Title**: Tên ngắn.
- **Status**: Proposed | Accepted | Deprecated | Superseded.
- **Context**: Tình huống và vấn đề.
- **Decision**: Quyết định (mô tả rõ ràng).
- **Consequences**: Lợi ích và trade-off.

Đánh số: `0001-`, `0002-`, … (số có leading zeros để sort đúng).

Tham chiếu: [21 Field-Based Simulation Architecture](../system/21-field-simulation-architecture-and-roadmap.md).
